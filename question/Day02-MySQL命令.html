一、MySQL的常见命令

1.1 查看当前所有数据库 

show databases;

1.2 指定当前库 

use 库名;

1.3 查看当前库的所有表 

show tables;

1.4 查看其它库的所有表 

show tables from 库名;

1.5 创建表 

create table 表名(
  列名 列类型,
  列名 列类型,
  ...
);

1.6 查看表结构 

desc 表名;

1.7 查看服务器的版本 

# 登录到mysql服务器
select version();

# 没有登录
mysql --version
# 或
mysql --V

二、MySQL的语法规范

  • 不区分大小写，但建议关键字大写，表名、列名小写
  • 每条命令最好用分号结尾
  • 每天命令根据需要，可以进行缩进或换行
  • 注释
  • 单行注释：# 注释文字
  • 单行注释：-- 注释文字
  • 多行注释：/* 注释文字 */
